# Amazon Clone

A front-end clone of the Amazon homepage built using HTML and CSS.

This project was created to strengthen my understanding of web development fundamentals, responsive layouts, and UI design by recreating the look and feel of Amazon's landing page.

## Features

- Amazon-inspired homepage layout
- Navigation bar with search section
- Hero banner section
- Product category cards
- Responsive design principles
- Clean and structured user interface

## Technologies Used

- HTML5
- CSS3
- Font Awesome
